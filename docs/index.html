How It Works

Simple 4-Step Pipeline

From repo URL to full documentation suite in under 2 minutes.

1

Scan Repo

GitHub REST API fetches your file tree, source code snippets, and config files.

2

Local LLM

DistilGPT-2 runs entirely on-device via Hugging Face Transformers — no internet calls, no API keys.

3

Write Docs

Four document writers craft README, API docs, setup guide, and a Mermaid diagram from the output.

4

Auto Commit

GitHub Actions commits the generated docs back to docs_output/ and uploads a downloadable artifact.
